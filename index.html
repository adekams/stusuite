StuSuite is a 3-in-1 app that consists of:

Find A Book : An app that helps a user search for various books by supplying the name or an information about the book.

Expense Tracker : This app allows a user to keep track of their expenses with the use of an inbuilt calculator that finds the difference between budget and total expenses.

TODO List : A great app for users to keep track of their schedule. It allows addition of new entries as well as deletion of a completed task.
